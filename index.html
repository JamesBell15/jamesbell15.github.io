Personal Marketing Images Automatically!

Marketing images was a cross discipline project to realise the designers designs. We wanted to make a tool for users to be able to market their website (provided by Nearcut) and their own brand.
To achieve this I created a Sidekiq task that would screenshot their site using a 3rd party api, then take that screenshot and composite an image using empty templates the designer provided.
Composition was created using a ruby implementation of ImageMagick.

3D Graphics Engine!

For my 3D computer graphics module I was tasked to implement a graphics engine using the principles taught. This involved setting up the graphics pipeline for OpenGL, writing Fragment Shaders and Vertex Shaders.
I used JOGL (Java OpenGL) as through my degree I was taught Java to a high standard. It was more important to demonstrate ability to understand the principles of 3D computer graphics, than the complex behavior of C++ which is the industry standard.

  • Java
  • OpenGL

The Most Entertaining Game!

As voted by the sheffjam 9 committee



Play it here!

This was our second game jam at Sheffield, organised and run by the CompSoc. The theme for the jam was "Villains" and so we chose to base the game on the most villainous act to steal forty pies, that is as many as four tens! And that's terrible!

We (Ilya, Jon, Halle, and myself) used a kanban board to organise tasks and implemented an accelerated Agile approach to development. I was responsible for Sound engineering, Display text, Point system, and technical implementation of graphics.

In 48 hours we produced the most entertaining game of that year's game jam.
